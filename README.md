# 🤖 Rule-Based Chatbot using Python

## 📌 Overview

This project is a simple Rule-Based Chatbot developed in Python. The chatbot responds to predefined user inputs using conditional statements and provides basic conversational interactions.

## 🚀 Features

* Greets users with friendly responses
* Answers simple questions
* Handles basic conversations
* Exit functionality using keywords like `bye`, `exit`, and `quit`
* Beginner-friendly Python project

## 🛠️ Technologies Used

* Python 3

## ▶️ How to Run

1. Clone the repository.
2. Open the project folder.
3. Run the Python file:

```bash
python chatbot.py
```

## 💬 Supported Commands

* Hi
* Hello
* Hey
* How are you?
* What is your name?
* Who are you?
* Bye
* Exit

## 🎯 Learning Outcomes

* Functions in Python
* Conditional Statements (if-elif-else)
* Loops
* String Manipulation
* User Input Handling

## 📷 Sample Conversation

```text
🤖 Chatbot Started

You: hi
Bot: Hi! How can I help you?

You: what is your name
Bot: I am a simple rule-based chatbot.

You: bye
Bot: Goodbye! Have a nice day 👋
```
